# applippliportal — プロジェクト概要

## プロジェクト
React / Next.js ベースの Web アプリケーション

## エージェント体制

このプロジェクトは **3エージェント体制** で開発を進める。

| エージェント | ロール | 責務 |
|---|---|---|
| **マネージャー** | ユーザー対話・調整 | 要件整理、タスク分解、コーダー/テスターへの指示、成果物レビュー |
| **コーダー** | 実装 | Next.js コンポーネント・API ルート・ロジックの実装 |
| **テスター** | 品質保証 | ユニットテスト・統合テスト作成、バグ検出・報告 |

## 開発フロー

```
ユーザー
  └→ マネージャー（要件受信・タスク分解）
        ├→ コーダー（実装）
        └→ テスター（テスト作成・検証）
             └→ マネージャー（結果まとめ・ユーザーへ報告）
```

## 技術スタック
- フロントエンド: Next.js (App Router), TypeScript, Tailwind CSS
- テスト: Jest, React Testing Library
- パッケージマネージャー: npm

## ディレクトリ構成
```
applippliportal/
├── src/
│   ├── app/          # Next.js App Router ページ
│   ├── components/   # 再利用可能コンポーネント
│   ├── lib/          # ユーティリティ・ヘルパー
│   └── types/        # TypeScript 型定義
├── __tests__/        # テストファイル
├── .claude/
```
